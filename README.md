# KnowerLife

🧠 **Самая хорошая работа – это высокооплачиваемое хобби.**

Добро пожаловать в профиль KnowerLife на GitHub!  
Я **системный аналитик и full‑stack разработчик**, создаю цифровые продукты, которые приносят бизнесу прибыль.

Этот профиль объединяет мои open‑source проекты, телеграм‑ботов, утилиты для автоматизации и **новое портфолио**, доступное онлайн по адресу:

🌐 **[knowerlife.ru](https://knowerlife.ru/)** – персональный сайт-портфолио с интерактивными инструментами, кейсами из Роскосмоса и SMM‑проектами.

## 🚀 Портфолио на knowerlife.ru

На сайте реализованы:
- **Тёмная / светлая тема**, анимированный курсор, адаптивная вёрстка.
- **Секции**: компетенции, системный анализ, кейс Роскосмоса, портфолио сайтов и SMM.
- **Встроенные мини‑приложения**: генератор пароля, ToDo-лист, калькулятор, таймер, шифр Цезаря и другие.
- **Контактная форма** и виджет комментариев ВКонтакте.
- **SEO‑оптимизация**, Open Graph, JSON‑LD микроразметка.

Исходный код сайта открыт и находится в репозитории **[portfolio](https://github.com/KnowerLife/portfolio)** (добавьте репозиторий на GitHub).

## Обо мне

Я специализируюсь на создании веб‑приложений, автоматизации бизнес‑процессов и системном анализе. Мои проекты охватывают:

- 🔹 Разработку сайтов и интернет‑магазинов с интеграциями (1С, СДЭК, Kafka).
- 🔹 SEO‑продвижение и SMM‑стратегии, выведшие проекты на рост в 10 раз.

- 🔹 Создание Telegram‑ботов и утилит для macOS.

## Ключевые проекты

- **[portfolio](https://github.com/KnowerLife/portfolio)**  
  Исходный код сайта‑портфолио. Адаптивный дизайн, встроенные инструменты, микроразметка.  
  **Язык**: HTML, CSS, JavaScript

- **[SecretSanta](https://github.com/KnowerLife/SecretSanta)**  
  Полнофункциональный Telegram‑бот для игры "Тайный Санта" с напоминаниями, анонимными сообщениями и мультиязычной поддержкой.  
  **Язык**: Python

- **[VPN](https://github.com/KnowerLife/VPN)**  
  Генератор конфига Cloudflare WARP для AmneziaWG, AmneziaVPN и WireGuard.  
  **Язык**: Shell

- **[Calculator](https://github.com/KnowerLife/calculator)**  
  Telegram‑бот для расчёта общих покупок: ручной ввод, QR‑код чека или CSV‑файл. Генерирует отчёт в CSV.  
  **Язык**: Python

- **[MMOGAME](https://github.com/KnowerLife/MMOGAME)**  
  Telegram‑бот в жанре MMO‑игры внутри мессенджера.  
  **Язык**: Python

- **[SCHIPSGAME](https://github.com/KnowerLife/SCHIPSGAME)**  
  Азартная игра "Фишки/Сотки" для школьников — ностальгия и веселье.  
  **Язык**: Python

- **[socialnetwork](https://github.com/KnowerLife/socialnetwork)**  
  SocialBot — Telegram‑бот, превращающий мессенджер в социальную сеть.  
  **Язык**: Python

- **[SA](https://github.com/KnowerLife/SA)**  
  Памятка системного аналитика (Essential Guide) — полезный справочник.  
  **Язык**: HTML

- **[shadowsocksmanager](https://github.com/KnowerLife/shadowsocksmanager)**  
  Приложение для macOS для управления подключениями Shadowsocks.  
  **Язык**: Swift
